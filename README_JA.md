# delta_trace_db

## 注意
現在、本ソフトウェアは開発中です。
この注意書きが削除されるまでは使用できないことに注意してください。

## 概要
各操作（デルタという単位を使います）をログとして記録し、差異の追跡と管理を容易にするインメモリデータベースです。
日本の医療機関などを含む、特定の要件が必要な現場のために最適化されています。
本ソフトウェアは大規模での運用は想定していません。
本ソフトウェアは、小規模でデータ数が少なく、検索速度よりも利便性やコスト、セキュリティ等が重視される場合に有用です。
それ以外のケースでは従来のDBを検討してください。

## 使い方
pub.devのExampleタブをチェックしてください。

## サポート
基本的にサポートはありません。
もし問題がある場合はGithubのissueを開いてください。
このパッケージは優先度が低いですが、修正される可能性があります。

## バージョン管理について
それぞれ、Cの部分が変更されます。  
ただし、バージョン1.0.0未満は以下のルールに関係無くファイル構造が変化する場合があります。  
- 変数の追加など、以前のファイルの読み込み時に問題が起こったり、ファイルの構造が変わるような変更
  - C.X.X
- メソッドの追加など
  - X.C.X
- 軽微な変更やバグ修正
  - X.X.C
