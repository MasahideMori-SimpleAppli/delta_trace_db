# delta_trace_db

現在、本ソフトウェアは開発中です。

## 概要
「DeltaTraceDB」は、各操作を「デルタ」としてログに記録することで、変更の追跡と管理をシンプルに行えるインメモリデータベースです。  
日本の医療機関など、特定の要件が求められる現場に適した設計であり、特に小規模で低コストな運用を重視しています。  

本ソフトウェアは、少量のデータ管理とコスト効率、セキュリティの確保が優先される中小規模プロジェクトに適しており、  
高齢化による社会的負担軽減も念頭に置いて開発されています。  
大規模データや検索速度が求められる場合には、一般的なデータベースの利用を推奨します。

## バージョン管理について
それぞれ、Cの部分が変更されます。  
ただし、バージョン1.0.0未満は以下のルールに関係無くファイル構造が変化する場合があります。  
- 変数の追加など、以前のファイルの読み込み時に問題が起こったり、ファイルの構造が変わるような変更
  - C.X.X
- メソッドの追加など
  - X.C.X
- 軽微な変更やバグ修正
  - X.X.C
